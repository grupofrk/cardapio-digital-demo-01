# Cardápio Digital Demo 01

Cardápio digital fictício para restaurante, desenvolvido como projeto de portfólio do Grupo FRK.

## Link do projeto online

Acesse o cardápio publicado:

https://grupofrk.github.io/cardapio-digital-demo-01/

## Objetivo do projeto

Criar um cardápio digital simples, responsivo e funcional para restaurantes que desejam apresentar seus produtos e receber pedidos pelo WhatsApp.

## Funcionalidades

- Categorias de produtos
- Entradas
- Pratos principais
- Bebidas
- Descrição dos produtos
- Preços destacados
- Botões de pedido pelo WhatsApp
- Layout responsivo
- Rodapé institucional

## Tecnologias utilizadas

- HTML
- CSS
- Git
- GitHub Pages

## Status do projeto

Primeira versão funcional publicada.
